## Notes on Using These Resources

### For Tedim Glossing:
1. Start with **Henderson 1965** for basic morphological patterns
2. Use **VanBik 2009** for etymological connections and PKC reconstructions
3. Check **Button 2011** for Northern Chin patterns
4. Use **Weera 1998** for tone patterns

### For Morpheme Identification:
- Kuki-Chin languages have stem alternation (Stem I vs Stem II)
- Pronominal affixes mark person/number
- Tone often carries grammatical information
- Aspect markers are important
