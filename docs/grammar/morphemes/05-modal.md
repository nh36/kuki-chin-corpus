## -ding (IRR)

### Overview

| Property | Value |
|----------|-------|
| Form | -ding |
| Category | Modal suffix |
| Function | Irrealis - future, potential, hypothetical |
| Attestations | 19,963 (most frequent modal) |

### ZNC (2017/2018)

ZNC documents -ding as the primary irrealis/future marker.

**Function:** Marks events that have not (yet) occurred—future, hypothetical, or potential.

**Corpus Examples:**

`pai ding kei`
[≈Bible: pai ding kei]
go.I IRR 1SG
"I will go"

`bawl ding hi`
[≈Bible: bawl ding hi]
make IRR DECL
"will make"

### Combinatorial Patterns

-ding frequently combines with:
- -in (ERG): `ding-in` for irrealis nominal clauses
- -lo (NEG): `ding lo` "will not"
- -maw (Q): `ding maw` "will...?" (future question)

## -thei (ABIL)

### Overview

| Property | Value |
|----------|-------|
| Form | -thei / -theih (Form II variant) |
| Category | Modal suffix |
| Function | Abilitative - can, able to |
| Attestations | 662 |
| Etymology | < thei "know" |

### ZNC (2017/2018: §5.7.4, pp. 150-153)

**Definition:** Potential mood expressing ability or possibility.

**Stem sensitivity:**
- Form I verb + -thei → -thei
- Form II verb + -thei → -theih (with final glottal)

**EXAMPLE (332):**
`nau-ngek tu-thei-ta`
[≈Bible: nau-ngek tu-thei-ta]
baby sit.I-ABIL-PFV
"The baby can sit"

**EXAMPLE (333):**
`an nek-theih-ta`
[≈Bible: an nek-theih-ta]
food eat.II-ABIL-PFV
"Food is ready for eating"

### Etymology

ZNC notes that -thei has a lexical verb counterpart meaning "know":
- Lexical: `thei` "to know"
- Modal: `-thei` "able to, can"

### Corpus Examples

`khiat-thei` "interpret-ABIL" = "can interpret"
`kham-theih-zu` "forbid-ABIL-NEG" = "cannot forbid"

| Attestations | 216 |

### Corpus Examples

`cih-nop-na`
[≈Bible: cih-nop-na]
say-WANT-NMLZ
"desire to say"

Note: -nop often occurs with nominalizer -na.

### Comparison with -nuam

Both express desire, but may differ in:
- Register
- Nominalization patterns (-nop + -na is common)
- Intensity of desire

## -pah / -pak / -lawh (NEG.ABIL)

### Overview

Three suffixes express negative ability ("cannot"):

| Suffix | Attestations |
|--------|--------------|
| -pah | 46 |
| -pak | 13 |
| -lawh | 13 |

### ZNC (2017/2018)

These mark inability to perform an action.

### Corpus Examples

**-pah:**
`bawl-pah` "make-NEG.ABIL" = "cannot make"
`om-pah` "exist-NEG.ABIL" = "cannot exist"

**-pak:**
`om-pak` "exist-NEG.ABIL" = "cannot exist"
`tai-pak` "rebuke-NEG.ABIL" = "cannot rebuke"

**-lawh:**
`ngah-lawh` "get-NEG.ABIL" = "cannot get"
`siat-lawh` "spoil-NEG.ABIL" = "cannot spoil"

### Semantic Distinctions

The distinction between -pah, -pak, and -lawh requires further research. Possible factors:
- Phonological conditioning
- Semantic nuance (temporary vs. permanent inability)
- Register or dialect variation

## -ngei (EXP)

### Overview

| Property | Value |
|----------|-------|
| Form | -ngei |
| Category | Modal/aspect suffix |
| Function | Experiential - have done before |
| Attestations | 748 |

### ZNC (2017/2018)

**Function:** Marks that the subject has experience of the action.

### Corpus Examples

`su-sia ngei nawn kei`
[≈Bible: su-sia ngei nawn kei]
CAUS-bad EXP CONT NEG.EMPH
"not having done badly before"

`ngei-na`
[≈Bible: ngei-na]
EXP-NMLZ
"experience (noun)"

### Classification Note

-ngei straddles the boundary between aspect (experiential aspect) and modality (epistemic/evidential).

## Morphosyntactic Patterns

### Stacking

Modals can co-occur:
- `-ding` + `-ngei` = future + experiential
- `-thei` + `-ta` = ability + perfective

### Nominalization

Many modals take nominalizing suffix -na:
- `-nop-na` (WANT-NMLZ) = "desire"
- `-ngei-na` (EXP-NMLZ) = "experience"

### Negation

Modals can be negated:
- `-thei-zu` "ABIL-NEG" = "cannot"
- `-ding lo` "IRR NEG" = "will not"
