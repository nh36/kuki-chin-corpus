## Overview: Case System in Tedim Chin

Tedim Chin exhibits **ergative-absolutive alignment** with post-nominal case markers.

| Marker | Gloss | Function |
|--------|-------|----------|
| -in | ERG | Transitive subject |
| -ah | LOC | Location/destination |
| -a | GEN/INSTR | Genitive/instrumental |
| -pan | ABL | Source/ablative |
| -tawh | COM | Comitative ("with") |

**Note:** The absolutive (intransitive subject, transitive object) is unmarked.

## -in (ERG)

### Overview

| Property | Value |
|----------|-------|
| Form | -in (ZNC: -ìn) |
| Category | Case marker |
| Function | Ergative - marks transitive subject |
| Allomorphy | None documented |
| Sukte cognate | -in |

### Henderson (1965: pp. 41-70)

Henderson treats -in as a "phrase-final particle" in subjective nominal phrases, using structural rather than semantic terminology.

**Examples:**
- `Dahpa in` (I 1) "[Dahpa]"
- `a innkuan pih te in` (I 1/8) "his whole family"
- `innkuan pih te in a tai tai uh a` (I 8) "his whole family..."

**Note:** Henderson does NOT use the term "ergative"; she analyzes -in as a structural phrase-final particle.

### ZNC (2017/2018: §3.3.3.3.1)

**Definition:**
> "Ergative case marker occurs in a transitive sentence. It marks the subject of a transitive verb."

**Semantic role:** Agent of transitive verbs (A-function in ergative-absolutive alignment)

**EXAMPLE (150):**
`a pa-in a insak-te meithai khat ngai aih...`
[≈Bible: a pa-in a insak-te meithai khat ngai aih]
3POSS father-ERG 3POSS neighbor-PL widow one think love
"Her father loved their neighbor who is a widow..."

**EXAMPLE (151) - Intransitive (no ergative):**
`lei khat awm in lei sang mah~mah khat awm in`
[≈Bible: lei khat awm in lei sang mah~mah khat awm in]
bridge one exist CONJ bridge high RED one exist CONJ
"There was a bridge, a very high bridge was there"

Note: Intransitive subjects are unmarked (absolutive).

### Otsuka (2009)

> "The subject of the causative construction, i.e. causer, is marked with the ergative case marker -in."

Shows ergative marking extends to causers in complex predicates.

### Singh (2018: Sukte)

**Cognate:** -in (functionally ergative/nominative)

**Examples:**
- `kei-in vəsə kə mu hi` "I see the bird"
- `ko-in vəsə kə mu uʔ hi` "We see the bird"

### Cross-Source Comparison

| Feature | Henderson | ZNC | Singh (Sukte) |
|---------|-----------|-----|---------------|
| Form | -in | -ìn | -in |
| Analysis | Phrase-final particle | ERG case marker | NOM/ERG marker |
| Function | Subjective NP | Transitive subject | Transitive subject |

### Open Questions

1. Does -in also mark instruments (as in some ergative languages)?
2. What triggers optional vs. obligatory -in marking?

## -ah (LOC)

### Overview

| Property | Value |
|----------|-------|
| Form | -ah (ZNC: -àɁ / -ā) |
| Category | Case marker |
| Function | Locative - location/destination |
| Allomorphy | -àɁ (with glottal) ~ -ā (without glottal) |

### Henderson (1965: pp. 54-65)

Henderson treats -ah as a post-nominal particle in "locative nominal figures."

**Examples:**
- `inn ah` (I 20) "to his house"
- `pialkhang ah` (I 4) "in the inner room"
- `a kua ah` (I 32) "to his hole"
- `lampi ah` (II 1) "on the road"
- `a lo uh ah` (I 8) "to their fields"

**Intonation pattern (pp. 54-56):**
- `a khe lang khat ah` "to one of his legs" (II 3)
- Low-high pitch pattern, figure-final glottalization
- VZT speaker uses high pitch on -ah after preceding word

### ZNC (2017/2018: §3.3.3.3.3)

**Definition:**
> "The locative indicates location or destination"

**Semantic roles:**
- LOCATION: Static position ("in the place where...")
- DESTINATION: Goal of movement ("to the place...")

**EXAMPLE (154):**
`lampi-ah pai`
[≈Bible: lampi-ah pai]
road-LOC go.I
"go on the road"

**EXAMPLE (155):**
`inn-ah ciah`
[≈Bible: inn-ah ciah]
house-LOC return
"return home"

### Cross-Source Comparison

| Feature | Henderson | ZNC |
|---------|-----------|-----|
| Form | -ah | -àɁ / -ā |
| Gloss | — | LOC |
| Function | Locative figure | Location/destination |
| Allomorphy | None noted | Glottal variation |

## -a (GEN/INSTR/LOC)

### Overview

| Property | Value |
|----------|-------|
| Form | -a (tonal variants) |
| Category | Case marker / Postposition |
| Function | Genitive, instrumental, locative |
| Tone distinction | -á (high) = GEN; -à (low) = INSTR/LOC |

### Henderson (1965)

Henderson documents -a in various constructions but does not analyze it as a unified case marker.

### ZNC (2017/2018)

**Syncretism:** ZNC documents tonal distinction:
- High tone -á: Genitive ("of")
- Low tone -à: Instrumental/locative

**EXAMPLE (Genitive):**
`pa-a laibú`
[≈Bible: pa-a laibu]
father-GEN book
"father's book"

**EXAMPLE (Locative):**
`van-a`
[≈Bible: van-a]
sky-LOC
"in the sky"

### Open Questions

1. Is -a a single morpheme with tonal variants or two homophonous morphemes?
2. What is the relationship between locative -a and locative -ah?

## -pan (ABL)

### Overview

| Property | Value |
|----------|-------|
| Form | -pan (ZNC: -pàn) |
| Category | Case marker |
| Function | Ablative - source of motion |
| Sukte cognate | -pan |

### Henderson (1965)

Henderson documents -pan in source expressions but does not label it "ablative."

### ZNC (2017/2018: §3.3.3.3.4)

**Definition:**
> "The ablative marker indicates the source or origin of motion"

**EXAMPLE:**
`inn-pan pai`
[≈Bible: inn-pan pai]
house-ABL go.I
"go from the house"

**EXAMPLE:**
`Yangon-pan hong pai hi`
[≈Bible: Yangon-pan hong pai hi]
Yangon-ABL toward go.I DECL
"(Someone) came from Yangon"

### Singh (2018: Sukte)

**Cognate:** -pan

### Cross-Source Comparison

| Feature | ZNC | Singh (Sukte) |
|---------|-----|---------------|
| Form | -pàn | -pan |
| Gloss | ABL | ABL |
| Function | Source of motion | Source of motion |

## -tawh (COM)

### Overview

| Property | Value |
|----------|-------|
| Form | -tawh (ZNC: -tɔ̀Ɂ) |
| Category | Case marker / Postposition |
| Function | Comitative - accompaniment ("with") |
| Allomorphy | Glottal stop variation |

### Henderson (1965)

Henderson documents -tawh in accompaniment expressions.

**Example:**
`heì tɔ̀Ɂ` "with an axe" (example 428 from ZNC citing Henderson context)

### ZNC (2017/2018: §3.3.3.3.5)

**Definition:**
> "The comitative marker indicates accompaniment"

**EXAMPLE:**
`nang-tawh pai ding kei`
[≈Bible: nang-tawh pai ding kei]
2SG-COM go.I IRR 1SG
"I will go with you"

**EXAMPLE (428) - Instrumental use:**
`Genpu-in hei tawh guasing-kung phuk luat manin`
[≈Bible: Genpu-in hei tawh guasing-kung phuk luat manin]
Genpu-ERG axe COM guava-tree fell.I wood many
"Genpu felled the guava tree with an axe"

**Note:** -tawh can mark both comitative ("with someone") and instrumental ("with something").

### Cross-Source Comparison

| Feature | Henderson | ZNC |
|---------|-----------|-----|
| Form | -tɔ̀Ɂ | -tɔ̀Ɂ |
| Gloss | — | COM |
| Function | Accompaniment | Comitative/instrumental |

1. Is -tawh historically related to directional -toh?
2. What distinguishes instrumental -tawh from instrumental -a?

## Cross-Source Synthesis

### Case Marker Paradigm

| Case | Form | Function | Sukte Cognate |
|------|------|----------|---------------|
| ERG | -in | Transitive subject | -in |
| ABS | ∅ | Intrans. subj., trans. obj. | ∅ |
| LOC | -ah | Location/destination | -ah |
| GEN | -a (H) | Possession | — |
| INSTR | -a (L) / -tawh | Instrument | — |
| ABL | -pan | Source | -pan |
| COM | -tawh | Accompaniment | — |

### Alignment Type

Tedim Chin shows **ergative-absolutive** alignment:
- A (transitive subject) marked with -in
- S (intransitive subject) unmarked
- O (transitive object) unmarked

### Analytical Differences

| Feature | Henderson (1965) | ZNC (2018) |
|---------|------------------|------------|
| Terminology | "Particles" | "Case markers" |
| Framework | Structural | Semantic roles |
| Ergative analysis | Not explicit | Explicit |
| Case inventory | Not enumerated | 5-6 cases |
