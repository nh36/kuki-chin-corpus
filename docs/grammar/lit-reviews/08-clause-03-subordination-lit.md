# Literature Review: Subordination in Tedim Chin

This report compiles previous scholarship on clause combining and subordination in Tedim Chin.

---

## 1. Henderson (1965)

### 1.1 Sentence Structure

Henderson uses the structural unit of the **sentence**, which contains **phrases** (subjective S, predicative P, adjunctive A):

> "The simple sentences in the texts may be grouped into types according to the number and sequence of their constituent phrases."

Common patterns: AP, SP, APP, SPP, APPP, etc.

### 1.2 Sentence Particles

Henderson identifies two sentence particles:
- **hi** — marks conclusive sentences
- **leh** — marks inconclusive sentences

> "A sentence containing the sentence particle 'hi' may be termed conclusive; one containing the sentence particle 'leh' may be termed inconclusive."

> "An inconclusive sentence together with the following conclusive sentence constitutes a **compound sentence**."

### 1.3 Adverbial Conjunctions

Henderson identifies **adverbial conjunctions** as a distinct word class:

> "Adverbial conjunctions are not found with abrupt syllable closure" (p. 13)

The key temporal subordinator **ciangin** 'when' is used throughout her texts:
```
a tun ciangin
3SG arrive when
'when he arrived'
```

### 1.4 Adverbial Figures

> "The intonation of adverbial figures is always rising." (p. 55)

Adverbial figures link clauses together with rising pitch patterns.

## 2. Zam Ngaih Cing (2017/2018)

### 2.1 Clause Combination Framework (§7.5)

ZNC identifies four types of clause combination:
1. Adverbial clauses
2. Complement clauses  
3. Relative clauses
4. Co-ordinate clauses

### 2.2 Adverbial Clauses (§7.5.1)

> "An adverbial clause functions as a verb phrase or whole clause modifier (Payne 1997: 316, 317). As the name adverbial suggests, it has an adverbial function in the clause and are also termed as adjuncts."

#### 2.2.1 Temporal Clauses (§7.5.1.1)

Three temporal adverbial functions:

**Before** (-má [≈-ma] 'before'):
```
kà [≈ka]    án [≈an]    nék [≈nek]-má [≈ma]         ín [≈in]        kì [≈ki]-síl [≈-sil]       másà [≈masa]
1POSS food  eat.2-before   CONJ      REFL-wash.1  first
'Before I eat food, I will take bath first'
```

**After** (-xìt [≈-khit] 'after'):
```
ká [≈ka]    nà [≈na]    sèp [≈sep]-xìt [≈khit]       ciāŋ      tɔ̀ldām [≈tawldam]     út [≈ut]     ìŋ [≈ing]
1POSS thing work.2-after  TEMP      rest       want.1 1SG.RLS
'After I work, I want to rest'
```

**During** (-súŋ [≈-sung] 'within', -kál [≈-kal] 'between'):
```
ká [≈ka]    ɔ́m [≈awm]-lɔùɁ [≈loh]-kál [≈-kal]         ín [≈in]        ná [≈na]    kàp [≈kap]-keí    ɔú [≈o]
1POSS EXIST-NEG.2-between CONJ      2SG   cry.1-NEG  IMP
'While I was away, please don't cry'
```

#### 2.2.2 Manner Clauses (§7.5.1.2)

Manner is marked by **bàŋ [≈bang]** 'like' + **īn [≈in]**:
```
á     ŋèt [≈nget]   bàŋ [≈bang]   īn [≈in]        ŋàɁ [≈ngah]      lèɁ [≈leh]     út [≈ut]      ìŋ [≈ing]
3SG   ask.2 like  CONJ      receive  COND    want.1  1SG.RLS
'I want him/her to get it like the way he/she asks'
```

#### 2.2.3 Purpose Clauses (§7.5.1.3)

Purpose is marked by **díŋ [≈ding]** (irrealis marker) + **īn [≈in]**:
```
kà [≈ka]    pá [≈pa]      mèɁ [≈meh]   leí    díŋ [≈ding]    īn [≈in]     vák [≈vak]-xià
1POSS father  curry buy.1  PURP   CONJ   roam.1-away
'My father came out to buy vegetables'
```

#### 2.2.4 Reason Clauses (§7.5.1.4)

Reason is marked by **háŋ [≈hang]** + **īn [≈in]**:
```
dàmlɔùɁnā [≈damlohna] háŋ [≈hang]      īn [≈in]        sáŋnaūpáŋ [≈sangnaupang]  támpí [≈tampi]     sáŋ [≈sang]     kàɁ [≈kah]   tʰeì [≈thei]-lɔù [≈-lo]    ùɁ [≈uh]
sickness  reason   CONJ      student    many      school  attend able.1-NEG PL
'Due to sickness, many students could not attend classes'
```

Result marked by **mán [≈man]** + **īn [≈in]**:
```
ní [≈ni]    tʰūm [≈thum]-sūŋ [≈sung]     guàɁ  zùk [≈zuk]    mán [≈man]     īn [≈in]      tuí-lét [≈-let]
day   three-within rain  pour.2 result  CONJ    water-overflow
'There is flood because of three days rain'
```

#### 2.2.5 Simultaneous Clauses (§7.5.1.5)

Marked by **-súŋ [≈-sung]** 'within' + **ín [≈in]**:
```
ká [≈ka]    tùt [≈tut]-súŋ [≈sung]      ín [≈in]        xuā      ɔ́ŋ [≈awng]       miál    pàʔ [≈pah]
1SG   sit.2-within CONJ      village  toward   dark    soon
'While I was sitting, it soon became dark'
```

#### 2.2.6 Conditional Clauses (§7.5.1.6)

**Simple conditional** (-lèɁ [≈-leh]):
```
guàɁ  á     zùk [≈zuk]    lèɁ [≈leh]      kà [≈ka]      puán  pʰɔú [≈pho]-té [≈-te]    kɔ́t [≈kawt]    díŋ [≈ding]
rain  3SG   drizzle COND    1POSS   cloth spread-PL wet    PURP
'If it rains, my clothes will get wet'
```

**Hypothetical** (hì [≈hi] lèŋ [≈leng]):
```
naūpáŋ [≈pang]   xàt [≈khat]   hì [≈hi]    lèŋ [≈leng]       ū-té [≈-te]        tʰú [≈thu]    màn [≈man]-sm    níŋ [≈ning]
child    one   COP   COND      elder-PL    word   obey.2-plan 1SG.IRR
'If I were a child, I'll try to obey my elders'
```

**Counterfactual** (-lècìn [≈-lecin]):
```
ín [≈in]-àɁ [≈ah]       ɔ́m [≈awm]     hì [≈hi]-lècìn [≈-lecin]    ɔ́ŋ [≈awng]      mù [≈mu]     díŋ [≈ding]    naìvè [≈ve]
house-LOC   EXIST  COP-COND    toward  see.1  PURP   2SG.COP.IMP
'If you had been at home, you would have seen me'
```

**Concessive** (tà [≈ta]-lèɁ [≈-leh]):
```
nī [≈ni]    sá [≈sa]    tà [≈ta]-lèɁ [≈-leh]       zɔ̀ŋ [≈zawng]   paī   vé [≈ve]~vè [≈ve]   nì [≈ni]
sun   hot   PFV-COND     also   go.1  still   1PL.INCL
'Even if it's sunny also, we'll still go'
```

**Negative conditional** (-keí buāŋ lèɁ [≈leh]):
```
guàɁ  á     zùk [≈zuk]-keí      buāŋ    léɁ [≈leh]     puán    sɔ́p [≈sawp]     díŋ [≈ding]
rain  3SG   rain.2-NEG   unless  COND    cloth   wash.1  PURP
'Unless it rains, we'll wash cloths'
```

#### 2.2.7 Concessive Clauses (§7.5.1.6.6)

Marked by **háŋ [≈hang]** + **īn [≈in]**:
```
á     xè [≈khe]    á     nàt [≈nat]    háŋ [≈hang]    īn [≈in]     ámá [≈ama]-n      bɔ̀Ɂlúŋ [≈bawhlung]   suì     vè [≈ve]~vè [≈ve]
3POSS leg   3SG   pain.2 reason CONJ   3SG-ERG    ball      kick.1  still
'Although his leg is paining, he still kicks ball'
```

#### 2.2.8 Substitutive Clauses (§7.5.1.6.7)

Marked by **sáŋ [≈sang]** 'instead':
```
kɔú [≈ko]      ín [≈in]-àɁ [≈ah]        mèɁ [≈meh]   huàn   sáŋ [≈sang]      cì [≈ci]    ìn [≈in]
1PL.INCL house-LOC    curry cook.2 instead  say.1 CONJ
'instead of cooking curry at home, we say...'
```

## 3. Singh (2018): Sukte Grammar [Comparative]

### 3.1 Temporal Clauses (§8.1.4)

> "Temporal clause in Sukte is formed [with] the subordinating words such as mə [≈ma] 'before' and khit 'after' which is placed before the finite verb."

**After** (khit):
```
kihilna    mən [≈man]    khit    pai    in
class      finish after   go     IMP
'Go after class finish.'
```

**Before** (mə [≈ma]) + **ciaŋin [≈ciangin]** 'when':
```
john ən [≈an]    ne     mə [≈ma]      ciaŋin [≈ciangin] ə     tai    ta    hi
John rice  eat    before  when   3SG   run    PST   DECL
'John ran before eating rice.'
```

### 3.2 Conditional Clauses (§8.1.6)

> "The conditional clause in Sukte is expressed by suffixing leʔ [≈leh] to the verb root."

```
nəŋ [≈nang]    don    leʔ [≈leh]    əməʔ [≈amah]  ə      don    diŋ [≈ding]    hi
you    drink  COND   he    3SG    drink  IRL    DECL
'If you drink, he will drink.'
```

### 3.3 Desiderative Clauses (§8.1.5)

> "The desiderative clause in Sukte is expressed by using the desiderative verb deiʔ [≈deih]."

```
kei    frans    əʔ [≈ah]      kə [≈ka]     pai    deiʔ [≈deih]   hi
I      France   LOC     1SG    go     DES    DECL
'I wish to go to France.'
```

## 4. Synthesis

### 4.1 Subordination Markers

| Function | Tedim Chin | Sukte |
|----------|------------|-------|
| **Temporal 'when'** | ciangin | ciaŋin [≈ciangin] |
| **After** | -xìt [≈-khit] ciangin | khit ciaŋin [≈ciangin] |
| **Before** | -má [≈-ma] ín [≈in] | mə [≈ma] ciaŋin [≈ciangin] |
| **During** | -súŋ [≈-sung] ín [≈in] | — |
| **Purpose** | díŋ [≈ding] ín [≈in] | díŋ [≈ding] |
| **Reason** | háŋ [≈hang] ín [≈in] | — |
| **Result** | mán [≈man] ín [≈in] | — |
| **Manner** | bàŋ [≈bang] ín [≈in] | — |
| **Conditional** | lèɁ [≈leh] | leʔ [≈leh] |
| **Counterfactual** | lècìn [≈lecin] | — |
| **Concessive** | tàlèɁ [≈taleh] | — |
| **Unless** | -keí buāŋ lèɁ [≈leh] | — |
| **Instead** | sáŋ [≈sang] | — |

### 4.2 Key Patterns

**Common pattern**: SUBORDINATOR + **ín [≈in]/in** (conjunctive particle)

Most subordinate clauses in Tedim Chin are marked by:
1. A semantic subordinator (háŋ [≈hang] 'reason', bàŋ [≈bang] 'manner', díŋ [≈ding] 'purpose')
2. Followed by **ín [≈in]** (conjunctive marker)

**Conditional paradigm** (Zam Ngaih Cing's unique contribution):
- Simple: -lèɁ [≈-leh]
- Hypothetical: hì [≈hi] lèŋ [≈leng]
- Counterfactual: -lècìn [≈-lecin]
- Concessive: tà [≈ta]-lèɁ [≈-leh]
- Negative: -keí buāŋ lèɁ [≈leh]

### 4.3 Distinctive Findings

**Henderson's contribution**:
- First identification of **conclusive** (hi) vs. **inconclusive** (leh) sentence dichotomy
- Describes compound sentences as: inconclusive + conclusive sentence

**Zam Ngaih Cing's contribution**:
- Most detailed conditional paradigm (5 types)
- Full inventory of adverbial clause types
- Clear examples with morpheme glossing

**Singh's contribution**:
- Shows cognate subordinators in Sukte (ciaŋin [≈ciangin], khit, leʔ [≈leh])
- Documents desiderative clause type
