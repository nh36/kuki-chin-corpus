# Literature Review: Phonology and Orthography

## Overview

This report synthesizes previous scholarship on Tedim Chin phonology, with particular attention to the orthographic conventions used by different authors. Understanding these conventions is essential for interpreting data across sources.

## 1. Henderson (1965)

Henderson's description is based on fieldwork with speakers from Tiddim, Burma (Myanmar). Her phonological analysis is embedded within a grammar/text study rather than presented as a dedicated phonology.

### 1.1 Consonant Inventory

Henderson identifies **18 initial consonants** and **9 final consonants**:

**Syllable-initial:**
| Manner | Labial | Alveolar | Velar | Glottal |
|--------|--------|----------|-------|---------|
| Voiceless stops | p | t | k | |
| Voiced stops | b | d | g | |
| Aspirated stops | pʰ | tʰ | | | <!-- ≈Bible: ph th -->
| Fricatives | v | s, z | x | h |
| Affricate | | c | | |
| Nasals | m | n | ŋ | |
| Lateral | | l | | |

**Syllable-final:** /p, t, k, m, n, ŋ, l, lʔ, ʔ/
[≈Bible: lh]

**Key observations:**
- The velar fricative is written **kh-** but represents [x]
- Henderson notes variation: "Luce (1985) and Namkung (1996) list the voiceless dorsal fricative /χ/" (as cited in later sources)
- **c-** is an affricate pronounced "with the tongue-tip down, and with contact of the blade of the tongue in the alveolar region" [p. 9]
- **z-** "represents a voiced alveolar fricative, pronounced with the tip down" often palatalized before front vowels
- Final **-h** represents glottal stop [ʔ] "in the great majority of cases"
- Final **-lh** represents [lʔ] (lateral + glottal stop sequence)
[≈Bible: lh]

Henderson also notes **ʔw** (pre-glottalized labiovelar continuant) in limited contexts.
[≈Bible: w]

### 1.2 Vowel Inventory

Henderson's vowel system (extracted from orthographic conventions):

| Letter | Phonetic Value | Environment |
|--------|---------------|-------------|
| a | [aː] | open syllable |
| a | [a] or [aː] | before C (not w, h) |
| a | [a] | before -h |
| e | [eː] | open syllable |
| e | [e] or [eː] | before most C |
| e | [e] | before -h |
| i | [iː] | open syllable |
| i | [i] or [iː] | before most C |
| i | [i] | before -h |
| o | [ou] | open syllable, before -h |
| o | [o] | before other C |
| u | [uː] | open syllable |
| u | [u] or [uː] | before most C |
| u | [u] | before -h |

**Diphthongs:**
- ai, au: closing diphthongs [ai], [aːi], [au], [aːu]
- ei: closing diphthong [ei] or [eːi]
- eu: closing diphthong [eu] or [eːu]
- ia: opening diphthong [ia]
- iu: diphthong [iu] or [iːu]
- oi: short diphthong [oi]
- **aw**: represents [ɔː] (long half-open back rounded)
- **awh**: represents [ɔʔ] (short + glottal stop)
[≈Bible: awh]

**Triphthongs:** iai, iau, uai, uau

### 1.3 Tone System

Henderson recognizes **three tones**:

> "My informants, however, recognized three possible tones for monosyllables. These were called by VZT 'high', 'mid', and 'low'." [p. 13]

**Phonetic realizations:**
| Tone | Open/Long Syllables | Closed by Stop |
|------|---------------------|----------------|
| High (1) | rising pitch | high level or short high rising |
| Mid (2) | level pitch | level |
| Low (3) | falling pitch | low level or short low falling |

**Important:** "Tones are not indicated in the orthography" [p. 9]

Henderson notes: "in the great majority of cases short syllables closed by a stop, i.e. those ending in the spelling with '-h, -lh, -k, -t, -p' or '-wh', are pronounced with a low level pitch" [p. 13]

### 1.4 Vowel Length

Henderson observes inconsistent orthographic marking:
> "Occasionally, but not regularly, the long vowel is indicated by doubling of the letter, e.g. 'baan' arm" [p. 11]

## 2. Zam Ngaih Cing (2017/2018)

ZNC's grammar provides the most comprehensive modern phonological description, based on Tedim as spoken in Churachandpur district, Manipur (India). Her analysis uses IPA notation with explicit tone marking.

### 2.1 Consonant Inventory

ZNC identifies **19 consonant phonemes**:

| Place | Bilabial | Labiodental | Alveolar | Palatal | Velar | Glottal |
|-------|----------|-------------|----------|---------|-------|---------|
| Stop (vl. unasp) | p | | t | c | k | ʔ |
| Stop (vd. unasp) | b | | d | | g | |
| Stop (vl. asp) | pʰ | | tʰ | | | | <!-- ≈Bible: ph th -->
| Nasal | m | | n | | ŋ | |
| Fricative (vl.) | | v | s | | x | h |
| Fricative (vd.) | | | z | | | |
| Lateral | | | l | | | |

**Key differences from Henderson:**
- Explicitly includes glottal stop /ʔ/ as a phoneme
- Identifies /x/ as velar fricative (not /kʰ/ or /χ/)
[≈Bible: kh]
- Notes that /v/ is classified as voiceless fricative (unusual)

ZNC compares inventories [p. 25-26]:
> "Henderson (1965) identified 18 initial consonants and 9 final consonants... Thang (2001) listed 17 consonants... two equally important sounds, velar fricative /x/ and glottal stop /ʔ/ are not included in Thang (2001)"

**Consonant distribution:**
- All consonants except /ʔ/ occur initially
- Only /p, t, k, ʔ, m, n, ŋ, l/ occur finally
- Voiced stops, aspirates, and fricatives (/b, d, g, pʰ, tʰ, v, s, z, h, x, c/) are restricted to initial/medial
[≈Bible: ph th]

### 2.2 Vowel Inventory

ZNC identifies **5 vowel phonemes**:

| | Front | Back |
|---|-------|------|
| Close | i | u |
| Close-mid | e | |
| Open-mid | | ɔ |
| Open | a | |

**Vowel length** is phonemic:
> "Vowel length is distinguished in Tedim Chin where the meaning of words changes depending on whether the vowel is long or short" [p. 37]

Examples:
- sīŋ 'shake' vs. sī:ŋ 'ginger'
[≈Bible: sing si]
- kāl 'footstep' vs. kā:l 'week'
[≈Bible: kal ka]

**Diphthongs** (8 total): /ai, au, ei, eu, ia, iu, ɔi, ui/
[≈Bible: oi]

**Triphthongs** (4 total): /iai, iau, uai, uau/

### 2.3 Tone System

ZNC uses **diacritics** to mark three tones:
- **High tone**: acute (á) - e.g., zá 'hear'
[≈Bible: za]
- **Mid tone**: macron (ā) - e.g., zā 'medicine/cigarette'  
[≈Bible: za]
- **Low tone**: grave (à) - e.g., zà 'hundred'
[≈Bible: za]

**Tonal restrictions:**
> "The Stem 1 verbs can carry either High, Mid or Low tones but Stem 2 can have only the High and Low tones" [p. 59]

ZNC documents a constraint on tone sequences:
> "Tedim Chin does not permit the combination of Mid + Low tone. So in places where the Low tone is expected to occur, it is usually replaced by a High tone" [p. 57]

Example: Expected *lisàk → Actual lisák 'cause to be tasty'
[≈Bible: lisak lisak]

**Grammatical tone:**
Tonal change marks possession on proper nouns:
- thāŋpū 'Thangpu (name)' → thāŋpú 'Thangpu's' [p. 60]
[≈Bible: thangpu thangpu]
- cīŋnɔ̄ 'Cingno' → cīŋnɔ́ 'Cingno's'
[≈Bible: cingnaw cingnaw]

### 2.4 Syllable Structure

ZNC classifies Tedim Chin syllable structure as "moderately complex":

**Basic template:** (C)V(C)

**Syllable types documented:**
- Monosyllables: V, VV, VC, CV, CVV, VVC, CVC, CVVC, CVCC, CVVV
- Disyllables: VV$CV, CV$CV, VC$CVV, etc.
- Trisyllables and quadrisyllables (through compounding/affixation)

**Constraints:**
- No consonant clusters in onset
- Only /l/ + /ʔ/ permitted as coda cluster (rare): kìlʔ 'lock', vàlʔ 'swallow'
[≈Bible: kilh valh]
- Final consonants are unreleased [p⌝, t⌝, k⌝]

### 2.5 Phonological Processes

**Gemination** occurs at syllable boundaries in disyllables:
- Stops: ìppí 'sack', mìttɔ̀ 'blind', síkkeu 'spoon'
[≈Bible: ippi mittaw sikkeu]
- Nasals: liàmma 'wound', dìnna 'standing place', máŋŋìlʔ 'forget'
[≈Bible: liamma dinna mangilh]
- Lateral: guallèl 'defeated'
[≈Bible: guallel]

**Glottalization** marks Stem 2 verbs:
- laú (Stem 1) → laùʔ (Stem 2) 'afraid'
[≈Bible: lau lauh]

**Vowel coalescence** in compounds and affixation.

## 3. Singh (2018) - Sukte/Salhte

Singh's grammar describes Sukte (Salhte), a closely related Kuki-Chin variety spoken in Manipur. The phonological system is very similar to Tedim.

### 3.1 Consonant Inventory

Singh identifies **19 consonant phonemes**:

| Manner | Bilabial | Labio-dental | Alveolar | Palatal | Velar | Glottal |
|--------|----------|--------------|----------|---------|-------|---------|
| Stop (unasp) | p, b | | t, d | | k, g | ʔ |
| Stop (asp) | pʰ | | tʰ | | kʰ | | <!-- ≈Bible: ph th kh -->
| Fricatives | | v | s, z | | | h |
| Affricate | | | | c | | |
| Nasals | m | | n | | ŋ | |
| Lateral | | | l | | | |

**Key difference:** Singh includes aspirated velar /kʰ/ rather than fricative /x/
[≈Bible: kh]
> "There are nineteen consonant phonemes found in the language" [p. 42]

### 3.2 Vowel Inventory

Singh identifies **6 vowel phonemes** (one more than ZNC):

| | Front | Central | Back |
|---|-------|---------|------|
| Close | i | | u |
| Close-mid | e | | o |
| Open-mid | | ə | |
| Open | | a | |

The schwa /ə/ and rounded /o/ distinguish Sukte from Tedim orthography:
- Sukte: /kə/ '1st person pronoun' vs. Tedim: usually written *ka*
[≈Bible: ka]
- Sukte: /hon/ 'crowd' vs. Tedim: uncertain correspondence

### 3.3 Tone System

Singh identifies **three tones** marked with diacritics:
- **High tone**: (´) - e.g., nái 'watch'
[≈Bible: nai]
- **Mid tone**: (unmarked) - e.g., nai 'near'
- **Low tone**: (`) - e.g., nài 'pus'
[≈Bible: nai]

Minimal triplets demonstrate contrast:
- zám 'reach by hand' / zam 'gong' / zàm 'creep or crawl' [p. 66]
[≈Bible: zam zam]
- lél 'just like that' / lel 'lose' / lèl 'a kind of basket'
[≈Bible: lel lel]

### 3.4 Syllable Structure

Like Tedim, Sukte has (C)V(C) syllable structure:
- Onset: single consonant or zero
- Peak (nucleus): vowel or diphthong
- Coda: optional consonant

**Critical note:** The Bible orthography (the "model orthography" per ZNC §2.4.2) does not mark tone at all. This is consistent with Henderson's observation that practical orthography lacks tone marking.

### 5.5 Key Orthographic Conventions in the Bible

Based on corpus sampling:

1. **Velar fricative/aspirate**: Written **kh** (khawm 'collect', khat 'one' but khempeuh 'all')

2. **Velar nasal**: Written **ng** (bangin 'like', lungdam 'glad')

3. **Open-mid back vowel**: Written **aw** (aw 'oh!', bawl 'do', kawm 'near')

4. **Glottal stop**: Written as final **-h** in many contexts (ahih 'it being', cih 'saying')
   - But often not written where ZNC would transcribe /ʔ/
   - Compare: ZNC neìʔ 'have' vs. Bible *neih* or *nei*
   [≈Bible: neih]

5. **Vowel length**: Generally not distinguished orthographically
   - ZNC notes this as an unresolved issue: "Some people used single vowel whereas some others used double vowels" [§2.4.2.1]

6. **Compounds**: Writing varies; no systematic rules
   - ZNC: "It is not clearly differentiated in the orthography as to what type of morpheme should be written together and which ones to be separated" [§2.4.2.3]

### 5.6 Converting Between Systems

**To normalize ZNC IPA → Bible orthography:**
- ʔ → h (word-finally) or Ø (medially)
- x → kh
- ŋ → ng
- ɔ → aw
- Remove tone diacritics
- lʔ → lh
[≈Bible: lh]

**To interpret Bible orthography → ZNC IPA:**
- kh → x (initial) or context-dependent
- ng → ŋ
- aw → ɔ
- -h → ʔ (when word-final after vowel)
- -lh → lʔ
[≈Bible: lh]
- Add tone from dictionary/context (unpredictable from spelling)

## 6. Synthesis: Points of Agreement and Divergence

### 6.1 Consonants

**All sources agree on:**
- Voiceless unaspirated stops: /p, t, k/
- Voiced stops: /b, d, g/
- Aspirated bilabial and alveolar: /pʰ, tʰ/
[≈Bible: ph th]
- Fricatives: /s, z, v, h/
- Palatal affricate: /c/
- Nasals: /m, n, ŋ/
- Lateral: /l/

**Point of divergence:** The velar aspirate/fricative
- Henderson: writes "kh" but describes it as [x] (velar fricative)
- ZNC: /x/ (velar fricative), explicitly contrasts with Thang's /kʰ/
[≈Bible: kh]
- Singh (Sukte): /kʰ/ (aspirated stop)
- Bible: writes "kh" (phonetic value underdetermined)

This variation may reflect:
1. Dialectal differences (Burma vs. India varieties)
2. Diachronic change
3. Analytic differences (fricative vs. aspirate interpretation)

### 6.2 Vowels

**All sources agree on:**
- Five core vowels: /i, e, a, u/ + one back rounded vowel
- Diphthongs and triphthongs
- Phonemic vowel length (variably marked)

**Point of divergence:** Back vowel analysis
- Henderson: "aw" [ɔː], "o" [ou] (as diphthong)
- ZNC: /ɔ/ (open-mid back), no /o/
- Singh: /o/ and /ə/ as distinct phonemes

### 6.3 Tone

**All sources agree on:**
- Three-way tonal contrast
- Termed "high," "mid," and "low"
- Not marked in practical orthography

**Point of divergence:** Phonetic realization
- Henderson: describes pitch contours (rising/level/falling)
- ZNC: treats as abstract tonemes with diacritics
- Singh: minimal pairs demonstrate contrast clearly

## 7. Open Questions

1. **Is the velar obstruent /x/ or /kʰ/?** Dialectal investigation needed.
[≈Bible: kh]

2. **How systematic is vowel length?** ZNC treats it as phonemic; practical orthography doesn't mark it consistently.

3. **What is the status of /o/ vs. /ɔ/?** Sukte seems to distinguish them; Tedim sources vary.

4. **Glottal stop distribution:** When is final -h pronounced as [ʔ] vs. omitted?

5. **Tonal sandhi rules:** ZNC documents some tone changes in compounds/phrases; systematic study needed.

6. **Stem 1/Stem 2 phonology:** The glottalization and tone restrictions on verb stems deserve detailed investigation (see verb stem reports).

---

## 8. Recommendations for This Project

For the morphological analyzer and grammar reports:

1. **Use Bible orthography** as the primary representation (practical, widely understood)

2. **Note IPA equivalents** when precision is needed, following ZNC conventions

3. **Do not assume tone** from spelling—it must be learned lexically or from context

4. **Treat kh as /x/** following ZNC's analysis (but note variation)

5. **Recognize -h ambiguity:** Final -h may represent glottal stop or be orthographic convention

6. **Accept length variation:** Treat VV spellings (when they occur) as equivalent to V
