# Tedim Chin Verb Transitivity and Argument Frames

This report documents verb argument structure based on corpus analysis.
Transitivity is classified by ERG marker frequency with the verb.

## Methodology

Tedim Chin has split-ergative alignment:
- **ERG** marker `-in` marks transitive agents
- **ABS** (unmarked) marks intransitive subjects and transitive patients

Classification thresholds (based on ERG frequency in corpus):
| Class | ERG% | Description |
|-------|------|-------------|
| TRANS | >40% | Canonical transitives |
| AMBI | 20-40% | Ambitransitive |
| INTR | <20% | Intransitive |

## Summary

Tedim Chin shows **fluid transitivity** with many ambitransitive verbs.
ERG marking is not strictly binary but gradient, correlating with semantic
properties of the event (agentivity, volitionality, affectedness).

| Class | Count | Verbs |
|-------|-------|-------|
| INTR | 8 | hi, om, pai, sih, nungta, ... (+3) |
| AMBI | 25 | mu, muh, za, zak, ci, ... (+20) |
| TRANS | 6 | ngai, dawt, en, hawl, bei, ... (+1) |
| DITRANS | 3 | pia, piak, pia(k)sak |
