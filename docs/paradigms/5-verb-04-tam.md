# Tedim Chin TAM Markers

This report documents the Tense-Aspect-Mood (TAM) suffix system in Tedim Chin.

## Overview

Tedim Chin expresses tense, aspect, and mood through verbal suffixes
attached after the verb stem. Multiple TAM markers can stack.

### TAM Suffix Template

```
VERB-STEM-(DERIV)-(ASPECT)-(DIRECTIONAL)-(MODAL)
```

## Summary

Tedim Chin TAM system features:

1. **Core aspectual markers**: -ta (PFV), -zo (COMPL), -kik (ITER)
2. **Modal markers**: -ding (IRR), -thei (ABIL)
3. **Continuative/Prospective**: -nawn (CONT), -lai (PROSP)
4. **Stacking**: Multiple markers can combine in fixed order
