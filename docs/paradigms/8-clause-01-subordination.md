## Verb Form Distribution by Clause Type

Henderson (1965) describes two verb stem forms:
- **Form I** (indicative): Main clause, conclusive sentence
- **Form II** (subjunctive): Subordinate clause, before relativizer

| Clause Type | Expected Form |
|-------------|---------------|
| main (final) | Form I |
| main (non-final) | Form II |
| temporal | Form II |
| causal | Form II |
| conditional | Form II |
| purpose | Form II |
| relative | Form II |

## Summary

Tedim Chin clause structure shows typical SOV/verb-final patterns with:

1. **Subordinators appear clause-finally** - e.g., *ciangin* 'when' marks
   the end of a temporal clause

2. **Rich subordination system** - temporal (*ciangin*), causal (*hangin*),
   conditional (*leh*), purpose (*dingin*), comparative (*bangin*)

3. **Verb form alternation** - Form II in subordinate clauses,
   Form I in final main clauses

4. **Clause chaining** - Multiple subordinate clauses can chain before
   the main clause, creating complex sentences
