# Tedim Chin Switch Reference and Clause Chaining

This report documents switch reference patterns (same-subject vs different-subject)
and clause chaining constructions in the Tedim Chin Bible corpus.

## Background

Switch reference is a grammatical system found in many Tibeto-Burman languages
that tracks whether the subject of a subordinate clause is the same as (SS)
or different from (DS) the subject of the main clause.

In Tedim Chin:
- **Same-subject (SS)**: Marked by converb suffix `-in` on the first verb
- **Different-subject (DS)**: No `-in` suffix; different agreement markers

---

## Same-Subject (SS) Converb Construction

The SS converb is formed with `-in` suffix on the verb stem.
It indicates that the subject of the converb clause is identical to
the subject of the following main clause.

### Pattern

```
SUBJ  [VERB-in]  [VERB]  hi
      (converb)  (main)
```

## Clause Chaining

Tedim Chin allows multiple converb clauses to chain before the main clause.
Each converb clause maintains same-subject reference with the final main clause.

### Chain Length Distribution

| Chain Length | Sentences | Percentage |
|--------------|-----------|------------|
| 1 converb(s) | 8,302 | 83.6% |
| 2 converb(s) | 1,433 | 14.4% |
| 3 converb(s) | 167 | 1.7% |
| 4 converb(s) | 25 | 0.3% |

**Total sentences with converb chains:** 9,935

## Different-Subject (DS) Patterns

When the subordinate clause has a different subject from the main clause,
the converb suffix `-in` is not used. Instead:

1. **Explicit subject NPs** appear in each clause
2. **Agreement markers** differ between clauses
3. **Subordinators** like *ciangin* 'when' mark clause boundaries

### Common DS Pattern: *ahih ciangin*

The construction *ahih ciangin* (be.3SG.REL when) is a common DS marker.
It introduces a temporal clause with a different subject from the main clause.

**Occurrences:** 1,482

### DS Examples

**Genesis 1:21**
> Tua ahih ciangin Pasian in tuipi ganhing golpite le amau nam ciatin tuipi sungah tampi...
> *Tua ahih ciang-in Pasian in tui-pi ganhing gol-pi-te le amau nam ciatin tui-pi sung-ah tampi*
> that be.3SG.REL then-ERG PASIAN ERG water-big animal creature-great-PL and 3PL.PRO nation each.kind.ERG water-big inside-LOC many
> KJV: *And God created great whales, and every living creature that moveth, which the waters brought forth ...*

## Summary

Tedim Chin exhibits a **converb-based switch reference system**:

| Feature | Same-Subject (SS) | Different-Subject (DS) |
|---------|-------------------|------------------------|
| Verb form | VERB**-in** | VERB (no -in) |
| Subject marking | Once (on main verb) | Explicit in each clause |
| Agreement | Shared | Different per clause |
| Subordinator | Optional | Often *ciangin*, *hangin* |

### Key Findings

1. **SS converbs are frequent**: 13,231 instances
2. **Clause chains common**: Most chains are 1-2 converbs (9,735 sentences)
3. **DS marked structurally**: Through subordinators and explicit subjects, not a dedicated morpheme

This system allows Tedim Chin to efficiently express complex multi-clause
sentences while clearly tracking referential coherence across clauses.
