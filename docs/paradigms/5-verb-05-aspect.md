# Aspect Suffixes in Tedim Chin

Aspect suffixes mark the internal temporal structure of events: whether complete, ongoing, repeated, etc.

## Inventory

| Suffix | Gloss | Description | Attestations |
|--------|-------|-------------|--------------|
| -kik | ITER | Iterative - again, repetition | 1,632 |
| -nawn | CONT | Continuative - still, ongoing | 1,023 |
| -khin | IMM | Immediate - already, just now | 1,012 |
| -ta | PFV | Perfective - completed action | 974 |
| -zo | COMPL | Completive - successfully done | 740 |
| -to | CONT | Continuative - ongoing | 145 |
| -mang | COMPL | Completive - completely | 132 |
