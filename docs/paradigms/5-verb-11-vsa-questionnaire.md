# Verb Stem Alternation Report: Tedim Chin

This report provides corpus attestations matching the VSA questionnaire format
(Zakaria Muhammad). Examples are drawn from the Tedim Chin Bible and categorized
by syntactic context.

## Syntactic Contexts

| Context | Description | Typical Stem Form |
|---------|-------------|-------------------|
| Affirmative | Main clause, positive polarity | Form I |
| Negative | Contains negation marker (lo, kei) | Form I |
| A/S Relative | Subject/agent relative clause | Form II |
| P Relative | Patient/object relative clause | Form II |
| Subordinate | Temporal, conditional, causal | Form II |
