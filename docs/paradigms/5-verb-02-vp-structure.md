# Tedim Chin Verb Phrase Structure

This report documents the structure of verb phrases in Tedim Chin,
including the verb template, serial verbs, auxiliaries, and particles.

## Verb Phrase Template

The Tedim Chin VP has the following general structure:

```
(NEG) + (OBJ.AGR) + SUBJ.AGR + VERB-STEM + (DERIV) + (ASPECT) + (DIR) + (TAM) + (SFP)
```

### Slot Descriptions

| Slot | Content | Examples |
|------|---------|----------|
| NEG | Negation | kei (NEG) |
| OBJ.AGR | Object agreement | hong- (3>1/2), kong- (1>3) |
| SUBJ.AGR | Subject agreement | ka-, na-, a-, i- |
| VERB-STEM | Lexical verb | pai, bawl, mu, gen |
| DERIV | Derivational | -sak (CAUS), -pih (APPL), ki- (REFL) |
| ASPECT | Aspect markers | -zo (COMPL), -kik (ITER) |
| DIR | Directional | -khia (out), -lut (in), -toh (up) |
| TAM | Tense-Aspect-Mood | -ding (IRR), -ta (PFV), -lai (PROSP) |
| SFP | Sentence-final particle | hi, hiam, hen, aw |

---

## Form I vs Form II

Tedim Chin verbs have two stems that alternate based on clause type:

| Feature | Form I (Indicative) | Form II (Subjunctive) |
|---------|---------------------|----------------------|
| Clause type | Main, conclusive | Subordinate, inconclusive |
| Shape | Open syllable | Closed syllable |
| Example | mu 'see' | muh 'see.II' |
| Example | nei 'have' | neih 'have.II' |

---

## Serial Verb Constructions

Tedim Chin uses serial verb constructions where multiple verbs
share a subject and describe aspects of a single event.

### Common Patterns

| Pattern | Example | Meaning |
|---------|---------|---------|
| V1 + V2(motion) | lak-khia | take-go.out 'take out' |
| V1 + V2(direction) | pai-lut | go-enter 'go in' |
| V1 + V2(result) | bawl-zo | make-complete 'finish making' |
| V1 + V2(manner) | pai-zawm | go-continue 'keep going' |

## Auxiliary Verbs

Several verbs function as auxiliaries in VP structure:

| Auxiliary | Meaning | Function |
|-----------|---------|----------|
| om | be, exist | Existential, progressive |
| hi | be (copula) | Equational, identificational |
| thei | can, able | Modal (ability) |
| nuam | want | Modal (volition) |
| ding | will | Modal (future/intention) |
| kul | must | Modal (obligation) |

## Sentence-Final Particles

Tedim Chin uses sentence-final particles for illocutionary force:

| Particle | Function | Gloss |
|----------|----------|-------|
| hi | Assertive | SFP |
| hiam | Interrogative (yes/no) | Q |
| hen | Imperative (polite) | IMP |
| ta | Imperative | IMP |
| aw | Exclamative | EXCL |
| lo | Imperative (hortative) | HORT |
| leh | Conditional consequence | then |

## Negation

Negation in Tedim Chin uses *kei* (or variant forms) before the verb:

```
ka-pai kei → I don't go
NEG position: before verb complex or at end
```

### Negative patterns

| Pattern | Example | Meaning |
|---------|---------|---------|
| V kei | pai kei | not go |
| V lo | pai lo | not go (milder) |
| om kei | om kei | not exist |
| thei kei | thei kei | cannot |

## Summary of VP Structure Types

### Type 1: Simple VP
```
SUBJ.AGR + VERB + (TAM) + (SFP)
a-pai-hi 'he goes'
```

### Type 2: Transitive VP with Object Agreement
```
OBJ.AGR + SUBJ.AGR + VERB + (TAM) + (SFP)
kong-mu-hi 'I see him'
```

### Type 3: Derived VP
```
SUBJ.AGR + VERB + DERIV + (TAM) + (SFP)
ka-pai-sak-hi 'I make go (send)'
```

### Type 4: Negated VP
```
SUBJ.AGR + VERB + NEG + (SFP)
ka-pai-kei-hi 'I don't go'
```
