# Tedim Chin Noun Paradigms

## Overview

Tedim Chin nouns can be marked for:
1. **Case**: Ergative (-in), Locative (-ah), Ablative (-panin), Comitative (-tawh)
2. **Number**: Plural (-te)
3. **Case + Number combinations**: e.g., N-te-ah (PL-LOC)

**Note on possession**: Unlike many related languages, Tedim Chin does NOT prefix possessive pronouns to nouns.
Instead, possession is expressed with separate pronouns: *ka khut* (my hand), *na khut* (your hand), *a khut* (his/her hand).
