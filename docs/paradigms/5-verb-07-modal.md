# Modal Suffixes in Tedim Chin

Modal suffixes express modality: possibility, ability, necessity, desire, or epistemic stance.

## Inventory

| Suffix | Gloss | Description | Attestations |
|--------|-------|-------------|--------------|
| -ding | IRR | Irrealis - future, potential | 19,963 |
| -ngei | EXP | Experiential - have done | 748 |
| -thei | ABIL | Abilitative - can, able to | 662 |
| -nuam | want | Desiderative - want to | 630 |
| -nop | want | Desiderative - desire | 216 |
| -mawk | perhaps | Dubitative - perhaps | 99 |
| -pah | NEG.ABIL | Negative ability - cannot | 46 |
