# Tedim Chin Pronominal Marking

This report documents the verbal agreement system in Tedim Chin.

## Overview

Tedim Chin verbs agree with their subjects through prefixes.
The system distinguishes person and number, with a special
inclusive/exclusive distinction in first person plural.

## Object/Inverse Prefixes

Tedim Chin uses portmanteau prefixes to mark both subject and object.

| Prefix | Function | Gloss | Example | Attestations |
|--------|----------|-------|---------|--------------|
| kong- | 1SG→3 | 1SG>3 | kong-mu (I see him/her) | 2512x |
| hong- | 3→1/2 | 3>1/2 | hong-mu (he sees me) | 15480x |
| ni- | 2→1 | 2>1 | ni-mu (you see me) | 0x |

### Direction System

The object prefixes encode a hierarchy: 1 > 2 > 3

- **Direct** (higher acts on lower): kong- 'I→him'
- **Inverse** (lower acts on higher): hong- 'he→me'

## Summary

The Tedim Chin pronominal system features:

1. **Obligatory subject agreement** via prefixes (ka-, na-, a-, i-)
2. **Inclusive/exclusive distinction** in 1PL (i- vs ka-...-uh)
3. **Plural suffix -uh** combines with prefixes for plural subjects
4. **Inverse marking** with hong- for SAP objects of 3rd person subjects
5. **Direct marking** with kong- for 1SG acting on 3rd person
