# Literature Review Reports Plan

## Goal
Create literature review reports that parallel each of our existing corpus-based grammar reports, compiling what previous scholars have said on each topic with chronological organization, direct quotations, and examples.

## Available Sources (Chronological)

### 1. Henderson 1965 - "Tiddim Chin: A Descriptive Analysis of Two Texts"
- **Type**: Full reference grammar based on two narrative texts
- **Coverage**: Phonology, syllable structure, sentences, phrases, figures, words, affixes, particles
- **Status**: OCR text available in `literature/tedim-ctd/ocr/henderson_65a_combined.txt`
- **Key sections**: Nouns (p.62), Verbs (p.69), Verbal prefixes (p.102), Suffixes (p.103), Particles (p.103)

### 2. Weera 1998 - "Tiddim Chin Tone"
- **Type**: Tonal analysis
- **Status**: PDF available

### 3. VanBik 2009 - "Proto-Kuki-Chin" (STEDT)
- **Type**: Comparative/historical reconstruction
- **Relevant for**: Etymology of morphemes, reconstructed forms
- **Status**: PDF available in `literature/comparative/`

### 4. Zam Ngaih Cing 2018 - "A Descriptive Grammar of Tedim Chin" (PhD thesis)
- **Type**: Comprehensive modern grammar (700+ pages in 2 parts)
- **Coverage**: Phonology, Nouns/NP (Ch.3), Adjectives (Ch.4), Verbs/VP (Ch.5), Grammatical Relations (Ch.6), Clause Structure (Ch.7)
- **Status**: PDF available in `literature/tedim-ctd/new/`
- **Key sections**: 
  - Case (3.3.3.3): Ergative, Instrumental, Locative, Genitive, Ablative, Comitative, Dative
  - Numerals (3.3.3.5): Cardinals, Ordinals, Fractionals, Distributives
  - Verb Stems (5.2)
  - Aspect (5.6): Perfective, Completive, Progressive, Habitual, Durative
  - Mood (5.7): Imperative, Hortative, Necessity, Potential, Optative, Desiderative
  - Causatives (5.8.2.2, 6.6.1.1)
  - Applicatives (6.6.1.2): Benefactive, Comitative, Relinquitive
  - Clause Structure (Ch.7)

### 5. Otsuka Papers (2024) - Five specialized articles
- `Otsuka_Applicative_TiddimChin.pdf` - Applicative constructions (-sak, -piʔ, -san)
- `Otsuka_CausativeBenefactive_TiddimChin.pdf` - Causative and benefactive
- `Otsuka_DirectionalAffixes_TiddimChin.pdf` - Directional affixes
- `Otsuka_Voice_TiddimChin.pdf` - Voice
- `Otsuka_BurmeseLoanwords_TiddimChin.pdf` - Loanwords

### 6. Related Kuki-Chin Resources
- Sizang/Siyin grammar (Stern 1963) - closely related dialect
- VanBik - Grammaticalization in Kuki-Chin
- Button 2011 - Proto Northern Chin
- Peterson papers - KC tense/aspect, applicatives

## Approach

### For each report:
1. Read corresponding corpus-based report to understand our analysis
2. Extract relevant sections from Henderson 1965 OCR
3. Extract relevant sections from Zam Ngaih Cing 2018 (via pdftotext)
4. Check Otsuka papers if topic matches
5. Check comparative sources (VanBik, Button) for historical context
6. Compile chronologically with:
   - Section for each source
   - Direct quotations (block quotes for longer passages)
   - Examples with original glossing from source
   - Brief summary of key points
   - Notes on agreements/disagreements between sources

### Report format:
```markdown
# [Topic] - Literature Review

## Overview
Brief summary of how this topic is covered in the literature.

## Henderson 1965
[Quotations, examples, page references]

## Zam Ngaih Cing 2018
[Quotations, examples, chapter/page references]

## Otsuka [Year] (if applicable)
[Quotations, examples]

## Comparative Notes
[VanBik PKC reconstructions, cross-KC patterns]

## Summary
Key points of agreement/disagreement, gaps in coverage
```
